**PlantsVsUndead** — небольшой учебный 3D-проект на Unity в жанре защиты линии.  
Игрок размещает растения на грядках, чтобы остановить волны врагов и не дать им добраться до дома.

Проект создан для практики игровой логики, работы с объектами сцены, интерфейсом и базовыми механиками на **Unity** и **C#**.

---

## 🎮 Основные возможности

### Растения

- 🌻 **Подсолнух** — периодически создаёт ресурс для посадки новых растений.
- 🌱 **Горохострел** — автоматически атакует врагов в своей линии.
- 🌰 **Орех-защитник** — блокирует врагов и выдерживает большой урон.

### Враги

- 🐝 **Обычный враг** — базовый противник со сбалансированными характеристиками.
- ⚡ **Быстрый враг** — имеет меньше здоровья, но быстрее передвигается.
- 🐖 **Толстяк** — медленный противник с большим запасом здоровья.

### Игровые системы

- 📈 **3 волны врагов** со стартовой, промежуточной и финальной панелями.
- ❤️ **Система жизней** — если враг достигает дома, игрок теряет одну жизнь.
- 🛠 **Лопата** — позволяет удалять уже посаженные растения.
- 🧩 **Интерактивные грядки** — растения можно размещать только на свободных клетках.
- 🎯 **Автоматическая атака** — растение обнаруживает врагов в своей линии и стреляет по ним.

## 🕹 Как играть

1. Нажмите **Старт**, чтобы начать уровень.
2. Выберите растение на панели.
3. Установите растение на свободную грядку.
4. Используйте подсолнухи для получения ресурсов.
5. Размещайте атакующие и защитные растения, чтобы сдерживать врагов.
6. Используйте лопату, если нужно убрать растение.
7. Победите все волны, не дав врагам исчерпать запас жизней.

## 🚀 Запуск проекта

1. Склонируйте репозиторий:

```bash
git clone https://github.com/qwertynoqwerty/PlantsVsUndead3D.git
```

2. Откройте проект в **Unity 6.0+**.
3. Дождитесь импорта ассетов.
4. Откройте основную сцену проекта.
5. Нажмите **Play**.
